# 🏦 Bank Management System
A robust Backend application built with **C#** and **SQL Server**, following the **Layered Architecture** (Data Access, Business Logic, and Presentation Layers).

## 🚀 Key Features
- **Account Management:** Create, update, and manage bank accounts.
- **Transactions:** Secure deposit and withdrawal operations.
- **Data Integrity:** Using SQL Triggers and Stored Procedures for consistent data.
- **Modular Design:** Clear separation of concerns for better maintainability.

## 🛠️ Tech Stack
- **Language:** C#
- **Database:** SQL Server (T-SQL)
- **UI:** Windows Forms
- **Data Access:** ADO.NET
